# JSON 格式化助手（Chrome 插件）

一个本地使用的 Chrome 扩展：在任意网页选中一段文字后，右键点击 **JSON 格式化**，即可在新标签页中以折叠/展开的树形结构友好查看 JSON。

## 功能

- ✅ 右键菜单：选中文本后出现「JSON 格式化」
- ✅ 自动从混杂日志中提取第一个完整的 `{...}` 或 `[...]`
- ✅ 支持转义字符串（如 `\"key\":\"value\"`）的二次解析
- ✅ 树形视图：折叠/展开、节点数量、彩色高亮
- ✅ 一键复制格式化后的 JSON
- ✅ 一键查看原始文本

## 安装步骤（开发者模式加载）

1. 打开 Chrome，访问 `chrome://extensions/`
2. 右上角打开 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本目录 `json-formatter-extension`
5. 安装成功后即可在任意页面使用

## 使用方法

1. 在网页（如你的日志页面）上用鼠标选中一段包含 JSON 的文本
   - 比如 `{"id":"2057293847725146112","settleNo":"2057293847725146112",...}`
   - 即便前后还带有时间戳、日志级别、类名等多余字符也可以
2. 右键 → 点击 **JSON 格式化**
3. 新标签页会自动打开，并以彩色树形格式展示 JSON
4. 顶部按钮：
   - **全部展开 / 全部折叠**
   - **复制格式化结果**
   - **查看原文 / 查看树形**

## 目录结构

```
json-formatter-extension/
├── manifest.json     # 扩展配置（Manifest V3）
├── background.js     # 右键菜单 + 向当前页注入 overlay
├── overlay.js        # 在当前页面注入弹窗（Shadow DOM 隔离样式）+ JSON 解析与树渲染
└── README.md
```
